
# 🛒 Market System (Tkinter + Python)

Aplicação desktop desenvolvida em Python que simula um sistema de mercado com controle de estoque e carrinho de compras, utilizando Programação Orientada a Objetos e interface gráfica com Tkinter.

---

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em consolidar fundamentos essenciais de desenvolvimento, incluindo:

* Estruturação de sistemas com POO
* Separação entre lógica de negócio e interface
* Manipulação de estado (estoque e carrinho)
* Validação de entradas do usuário
* Integração entre backend e interface gráfica

---

## 🚀 Funcionalidades

* 📦 Visualização de produtos em estoque
* ➕ Adição de produtos ao carrinho por ID
* ➖ Remoção de produtos do carrinho
* 🧾 Exibição do carrinho com cálculo automático do total
* ⚠️ Validação de entrada (campos vazios e valores inválidos)
* 🔄 Atualização dinâmica da interface

---

## 🧠 Arquitetura

O projeto foi estruturado separando responsabilidades em diferentes camadas:

* `Produto` → entidade base com atributos e controle de estoque
* `Mercado` → gerencia o estoque e busca de produtos
* `Carrinho` → controla os itens adicionados e regras de compra
* `interface` → camada de apresentação (Tkinter)
* `main` → ponto de entrada e orquestração

Essa separação facilita manutenção, escalabilidade e reutilização do código.

---
