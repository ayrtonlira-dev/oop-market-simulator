# 🛒 Senna's Market

Simulador de mercado com carrinho de compras desenvolvido em Python, utilizando Programação Orientada a Objetos (POO) e interface gráfica com Tkinter.

---

##  Sobre o projeto

O **Senna's Market** é um sistema simples que simula o funcionamento de um mercado, permitindo:

* Visualizar produtos disponíveis no estoque
* Adicionar produtos ao carrinho
* Remover produtos do carrinho
* Visualizar o carrinho com cálculo automático do total

O projeto foi desenvolvido com foco em aprendizado e prática de conceitos fundamentais de programação.

---

## Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Separação de responsabilidades (classes independentes)
* Manipulação de estado (estoque e carrinho)
* Validação de entrada de dados
* Tratamento de erros (try/except)
* Interface gráfica com Tkinter
* Integração entre lógica e interface

---
## 🖥️ Interface

A aplicação possui uma interface simples onde é possível:

* Inserir ID do produto
* Inserir quantidade
* Adicionar ao carrinho
* Visualizar estoque
* Visualizar carrinho
