# 🛒 OOP - Market Simulator

Sistema de mercado desenvolvido em Python utilizando Programação Orientada a Objetos (POO).
O projeto simula operações básicas de um mercado, incluindo gerenciamento de estoque e carrinho de compras via terminal.

---

##  Funcionalidades

*  Cadastro de produtos
*  Gerenciamento de estoque
*  Carrinho de compras
*  Adicionar produtos ao carrinho
*  Remover produtos do carrinho
*  Cálculo automático do total da compra
*  Visualização detalhada do carrinho
*  Menu interativo no terminal

---

## Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento de dados
* Separação de responsabilidades entre classes
* Uso de dicionários para armazenamento eficiente
* Manipulação de fluxo com estruturas condicionais
* Tratamento de erros e validações básicas
* Integração entre múltiplas classes (`Produto`, `Mercado`, `Carrinho`)
